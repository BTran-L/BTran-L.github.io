OTAVA Learning

A comprehensive UX design work for Nova's new learning platform

Role UX Designer
Timeline 2019-2021
Platform Web Application

Project Overview

OTAVA Learning is Nova's new comprehensive learning platform designed to revolutionize the educational experience for both teachers and students. The platform focuses on streamlining daily teaching tasks while creating an inclusive learning environment accessible to all users.

The Challenge

Teachers were struggling with fragmented tools and inefficient workflows that consumed valuable time that could be spent on actual teaching. Students, particularly those with diverse learning needs, faced barriers in accessing educational content effectively.

My Role

As a UX Designer, I was responsible for:

  • Leading user research with teachers and students
  • Creating user flows and interaction designs
  • Developing high-fidelity prototypes
  • Conducting usability testing sessions
  • Collaborating with developers and product managers
  • Ensuring WCAG 2.1 AA accessibility compliance

Design Process

I followed a user-centered design process that emphasized continuous feedback and iteration. Through extensive shadowing sessions with teachers and interviews with students, I identified key pain points and opportunities for improvement.

Key Features Designed

Teacher Dashboard: A streamlined interface that provides quick access to daily tasks, student progress, and lesson planning tools.

Inclusive Content Delivery: Designed with accessibility at its core, featuring screen reader optimization, keyboard navigation, and customizable display options.

Real-time Collaboration: Tools that enable seamless communication between teachers, students, and parents.

Impact

The new platform improved teacher efficiency by reducing administrative time by 40% and increasing student engagement scores by 35%. Accessibility features ensured that 100% of students could access all learning materials effectively.

Lessons Learned

This project reinforced the importance of involving end users throughout the design process. Regular testing with diverse user groups led to insights that fundamentally improved the platform's usability and accessibility.
